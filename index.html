Sabemos que você é multitarefa, mas as leis da física e o código de trânsito discordam.

Este sistema foi projetado para ser operado por alguém que não esteja segurando um volante. Se você for o motorista, por favor, entregue o dispositivo a um passageiro ou aceite que a tecnologia ainda não cura colisões.

CONFIGURAÇÕES

O QUE É O SISTEMA?
O Elite Pro é um coletor de evidências georreferenciadas. Ele sincroniza o fluxo da câmera com a base de dados do SNV (Sistema Nacional de Viação) via GPS em tempo real.

COMO USAR:
1. Pasta: Ao iniciar, você define onde os PDFs serão salvos. O sistema grava sozinho lá.
2. Sincronia: O Km exibido no topo é extraído da sua posição GPS cruzada com o arquivo CSV carregado.
3. Captura: Ao clicar em INICIAR, o sistema entra em modo contínuo. Ele gera um frame com marca d'água (Km, Data, Hora) na base de cada foto.
4. Automação: Para evitar perda de dados, o sistema fecha um arquivo PDF automaticamente ao atingir o limite de fotos definido e inicia o próximo sem parar a captura.
5. Conclusão: Clique em FINALIZAR para fechar o lote atual e salvar o último arquivo.
