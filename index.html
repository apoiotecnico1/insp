GUIA DO SISTEMA

Este sistema realiza a video-inspeção rodoviária com georreferenciamento automático.

  • INICIAR: Começa a gravar e capturar fotos a cada 200ms.
  • GERAR PDF: Fecha o lote atual e salva o arquivo no celular.
  • CSV: Base de dados necessária (KM/LAT/LON) para atualizar a placa verde.
  • CONFIGURAÇÕES: Use para trocar entre lentes (Macro/Wide) e ajustar luz noturna.
